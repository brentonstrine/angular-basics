The http service

The http service can be used for basic HTTP stuff. Here $http.get('js/people.json') is used to AJAX retrieve a JSON file for use in the loop below.

myApp.controller('MyController', function MyController($scope, $http){
	$http.get('js/people.json').success(function(data){
		$scope.people = data;
	});
});
