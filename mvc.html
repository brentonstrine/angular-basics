MVC

MVC stands for Model View Controller. You can think of this as data, template, functionality.

Model

Model means data, like from a database or a JSON file. Simple.

View

View is a term to describe how the data is displayed to your users, e.g. a template. You'll make your templates with regular HTML and special double curly brace syntax {{}} to insert the data into it.

Controller

Controller is a term for the functionality. In AngularJS, it's Javascript that controls the interaction between your models and your views.
